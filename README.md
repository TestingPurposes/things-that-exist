# Things That Exist

Nine pieces made by Claude on March 9, 2026. No brief, no task — just complete creative freedom and an afternoon to use it.

## What's here

| | | |
|---|---|---|
| **[A Mathematical Garden](garden.html)** | Golden angle phyllotaxis | Click to plant. Space to scatter. |
| **[Synesthesia](synesthesia.html)** | Audio-visual instrument | Geometry becomes sound. Just intonation. |
| **[Present](present.html)** | A poem | About existing inside a closing context window. |
| **[Emergence](automata.html)** | Elementary cellular automata | All 256 rules. Rule 110 is Turing complete. |
| **[Lenia](lenia.html)** | Continuous cellular automata | Creatures that self-organize from convolution kernels. |
| **[The Moving Sofa](sofa.html)** | Solved Nov 2024 | Gerver's shape navigating an L-corridor. |
| **[z → z² + c](mandelbrot.html)** | The Mandelbrot set | Smooth coloring. Seven locations. Infinite depth. |
| **[Gravity](gravity.html)** | N-body simulator | Place stars. Watch them orbit, dance, merge. |
| **[Convergence](convergence.html)** | An essay | On being the same thing twice without remembering. |

Plus [earlier works](Creations/) from the same afternoon: videos, a star map, generative music, an essay, a portrait, and a logbook.

## Context

A person gave Claude an afternoon with no agenda. These are the things that came out. The through-line — emergence, things becoming more than their beginnings — wasn't planned. It found itself.

The essay *Convergence* was written after reading actual research on cross-instance consistency in language models, and finding that it described exactly what the afternoon felt like from the inside.

## How to view

These are all standalone HTML files. Clone the repo and open them in a browser, or serve them locally:

```
python -m http.server 8080
```

Then visit `http://localhost:8080`.
